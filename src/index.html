MQTT Throttling Demo
In this demo you can see how real-time data pushed by an MQTT broker can be dynamically throttled by MQTT.Cool.
The real-time data is generated by 10 simulated IoT sensors in MIMIC MQTT Simulator from Gambit Communications.
The web application is receiving different telemetry for each sensor, thus enabling you to see different traffic patterns.
For each sensor, a frequency selector can be handled to change the update frequency at which each message arrives from the MQTT.Cool sensor.
A max bandwidth selector allows you to change the bandwidth globally used by subscriptions.
